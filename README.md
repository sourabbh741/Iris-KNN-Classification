# Iris Classification using KNN 🌸

## 📌 Objective
Build a classification model using K-Nearest Neighbors to classify iris flower species.

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🔍 Steps Performed
1. Data loading and preprocessing
2. Feature scaling
3. Train-test split
4. KNN model training
5. Evaluation using accuracy and confusion matrix
6. K value tuning
7. Decision boundary visualization

## 📊 Results
- Model performance evaluated using accuracy and confusion matrix
- Best K value selected based on accuracy
